# هيكلة مشروع Java لقراءة pokedex_gen1.csv

## البنية:
pokedex_project/
 ├─ src/
 │   └─ pokedexapp/
 │        └─ MainReadCSV.java
 ├─ pokedex_gen1.csv
 └─ README.md

## طريقة التشغيل:
1. ضع ملف pokedex_gen1.csv في نفس مجلد المشروع.
2. Compile:
   javac src/pokedexapp/MainReadCSV.java
3. Run:
   java -cp src pokedexapp.MainReadCSV
